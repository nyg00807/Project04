## 📌 개요
인터랙션과 반응형 구조 설계를 중심으로 구현한 프론트엔드 포트폴리오 프로젝트입니다.
개발자가 누구인가에 대한 설명과 지금까지 작업한 프로젝트들을 담았습니다.

## 💬 회고
이번 포트폴리오를 제작하면서 단순히 결과물을 나열하는 것이 아니라,
한 명의 개발자가 어떤 고민을 거쳐 기획하고 디자인하며 구현했는지를 보여드리고 싶었습니다.

또한, 그동안 작업했던 프로젝트들을 돌아보며 단순한 기능 구현에 머무르지 않고,
왜 이런 구조를 선택했는지, 사용자에게 어떤 경험을 주고 싶은지에 대해 스스로 질문하는 과정을 거쳤습니다.

특히 레이아웃 구조 설계, 이벤트 흐름 제어, 반응형 전환 방식 등을 직접 고민하며
화면을 만드는 사람이 아니라 동작을 설계하는 개발자가 되고자 노력했습니다.
